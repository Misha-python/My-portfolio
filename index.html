Итак пишем (код выражается курсивом):
<!DOCTYPE html>
<html lang = 'ru' charset = 'utf-8'>
<head>
    < title>Пишем заголовок</title>
</head>

Этот код делает ПУСТУЮ страницу с заголовком 'Пишем заголовок'.

Чтобы сделать эффекты, линии, картинки используйте следующие команды:
<body>
    <em>Тут будет курсив!</em>
    <hr> - это будет черта:


    <img src = 'Полный или относительный путь к картинке.'>
    Текст можно писать прямо так...
    <p>Новый обзац</p>
    <br> Enter
    <strong> - жирный</strong>
    <s> - зачёркнутый текст</s>
    <u> - подчёркнутый текст </u>
</body>
</html>
